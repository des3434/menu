Barg Menu App

=== FREE APK CONVERSION GUIDE (Using GitHub Pages) ===

STEP 1: Create a GitHub Account (if you don't have one)
1. Go to https://github.com
2. Click "Sign up" and create a free account

STEP 2: Create a New Repository
1. Click the "+" icon in the top right corner
2. Select "New repository"
3. Name it: my-menu-menu
4. Make sure "Public" is selected
5. Check "Add a README file"
6. Click "Create repository"

STEP 3: Upload Your Menu Files
1. In your new repository, click "Add file" > "Upload files"
2. Drag and drop these files from the ZIP: index.html, manifest.json, sw.js, icon-192.png, icon-512.png
3. Click "Commit changes"

STEP 4: Enable GitHub Pages
1. Go to repository "Settings" (gear icon)
2. In the left sidebar, click "Pages"
3. Under "Source", select "Deploy from a branch"
4. Under "Branch", select "main" and "/ (root)"
5. Click "Save"
6. Wait 2-3 minutes for deployment
7. Your site URL will be: https://YOUR-USERNAME.github.io/my-menu-menu

STEP 5: Convert to APK (Free)
1. Go to https://www.pwabuilder.com
2. Paste your GitHub Pages URL
3. Click "Start"
4. Wait for analysis (usually 30 seconds)
5. Click "Package for stores"
6. Select "Android" (free option)
7. Click "Generate" and download the APK

=== CAN I INSTALL ON ANY DEVICE? ===
Yes! The APK works on any Android device (Android 5.0+).
Before installing, enable "Install from unknown sources" in your device settings.

=== ALTERNATIVE FREE OPTIONS ===
- Netlify Drop: https://app.netlify.com/drop (drag & drop your files)
- Vercel: https://vercel.com (connect your GitHub repo)
- Cloudflare Pages: https://pages.cloudflare.com

The project.json file contains your complete menu data for backup/restore.
